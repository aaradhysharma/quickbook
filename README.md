# Brew Haven Cafe — QuickBooks Alternative Cost Assignment

Excel / Google Sheets alternative to QuickBooks for categorizing fixed vs variable costs, projecting profitability, and running simple regression cost forecasts.

## Deliverables

| File | Purpose |
|------|---------|
| `Brew_Haven_Cafe_Cost_Tracker.xlsx` | Full workbook (startup, ongoing, ledger, P&L, regression) |
| `docs/Written_Report.md` | 400–500 word written report |
| `screenshots/` | Hands-on practice + regression output images |
| `index.html` | Viewable summary with embedded screenshots |

## Quick start

```bash
python build_assignment.py
```

Open `Brew_Haven_Cafe_Cost_Tracker.xlsx` in Excel or upload to Google Sheets. Open `index.html` in a browser for the report + figures.
